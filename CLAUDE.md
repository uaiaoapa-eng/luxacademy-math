# 럭스아카데미 수학관 홈페이지 — 프로젝트 핸드오프

> 이 파일은 Claude Code가 세션 시작 시 자동으로 읽습니다. 다른 컴퓨터/새 대화에서도 이 문서로 전체 맥락을 즉시 파악할 수 있도록 작성됨.

## 무엇인가
LuxAcademy_Storyboard(아임웹용 화면정의서 PDF)를 기반으로 만든 **정적(static) 멀티페이지 웹사이트**.
빌드 도구 없음 — 순수 HTML + CSS + JS. 어떤 PC에서도 `index.html`만 열면 동작.

## 페이지 구성
- `index.html` — 메인 원페이지 (Hero → 숫자 → 학년카드 → 비교표 → 7단계 → 강사진 → 실적탭 → 상승스토리/후기 → 신청폼 → FAQ/위치)
- `middle.html` `high.html` `sat.html` — 수학관 서브(중등/고등내신/수능)
- `admission.html` — 입학 안내·FAQ·신청폼
- `robot.html` — AI·로봇관 (ROBOFEST/WRO 국제 로봇대회 대비반)
- `legal.html` — 이용약관·개인정보처리방침·동의전문 (#terms/#privacy/#consent)
- `styles.css` — 전체 디자인 / `script.js` — 탭·아코디언·폼·모바일메뉴
- `assets/` — 모든 이미지 / `server.py` — 로컬 미리보기 서버 / `cloudflared.exe` — 임시 공개링크용

## 디자인 규칙
- **톤: 흰 배경 중심(라이트) + 파랑/하늘색 강조** (`--gold`=#2b8aef 가 실제로는 블루, `--gold-ink`=#1769c4). 변수명은 'gold'지만 값은 블루임(과거 골드 테마에서 변수명 유지).
- 폰트 Pretendard(CDN). 실적 강조는 녹색(`--green`) 유지.
- 모바일(~768px): 햄버거 메뉴, 하단 고정 CTA바, 7단계는 세로 타임라인, 비교표는 표 유지(카드 아님), 플로팅/헤더연락처 숨김.
